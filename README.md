 ML--ASSIGNMENT--5
 This repository presents two supervised machine learning classification models implemented using Python to solve real-world prediction problems.

The first project uses the K-Nearest Neighbors algorithm to predict whether a tumor is benign or malignant based on medical measurements from the Breast Cancer Dataset. The model uses features such as radius, texture, perimeter, area, and smoothness. The workflow includes data inspection, preprocessing, label encoding, and feature scaling, which is essential for KNN since the algorithm relies on distance calculations. The dataset is split into training and testing sets, and the classifier is trained while experimenting with different values of K to observe how the number of neighbors affects model performance. The results are evaluated using Accuracy, Precision, Recall, and F1 Score, along with analysis of misclassified cases and the model’s sensitivity to K. Visualizations include a confusion matrix, accuracy vs K plot, and decision boundary plots.

The second project implements a Decision Tree to predict whether a loan application should be approved or rejected using the Loan Prediction Dataset. The dataset includes features such as applicant income, loan amount, credit history, education level, and property area. Data preprocessing steps include handling missing values and encoding categorical variables before splitting the dataset into training and testing sets. A Decision Tree classifier is trained and analyzed by experimenting with tree depth and pruning techniques to understand how model complexity affects performance and potential overfitting. The model is evaluated using Accuracy, Precision, Recall, and F1 Score, and feature importance is examined to identify which factors most influence loan approval decisions. Visualizations include confusion matrices, tree structure plots, and feature importance graphs.

Together, these projects demonstrate the application of two important machine learning algorithms for medical diagnosis prediction and financial decision-making systems, highlighting the impact of preprocessing, hyperparameter tuning, and visualization in building interpretable classification models.
